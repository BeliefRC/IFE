表格插件&默认配置参数说明

    本插件可配选项:
  • 表格数据
  • 插入位置 (注意一个容器中不要插入两个表格会引起BUG)
  • 总体排序开关
  • 单列排序开关
  • 表头冻结
  • 表头背景色
默认参数&调用:

var table1 = TableTool.init({
    append:$('.body'),
    data:{
        thead:[],
        sortSwitch:[],
        tbody:{}
    }
    isSort:true,
    isFrozen:true,
    headColor:'defaultColor
}),
